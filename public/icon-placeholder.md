# PWA Icons

The app references these icon files:
- icon-192.png
- icon-512.png  
- icon-maskable-512.png
- apple-touch-icon.png
- favicon-32.png
- favicon-16.png

## To generate them:
```bash
npm install canvas
node scripts/generate-icons.cjs
```

Or replace with your own branded PNG files in /public/.
